# Milestone v1.0: MVP Launch

**Status:** ✅ SHIPPED 2026-03-01
**Phases:** 1–11
**Total Plans:** 38
**LOC:** ~16,500 TypeScript/TSX (121 files)
**Timeline:** 2026-02-11 → 2026-03-01 (18 days)

## Overview

Een volledig werkend AI-bijlesplatform waar Arubaanse kinderen (klas 1-6) zelfstandig bijles krijgen van Koko, de Socratische AI-tutor. Het platform omvat ouder-authenticatie, handmatige betalingsverificatie, beginsituatietoets per vak, voortgangsregistratie, een ouderportaal met wekelijkse e-mails, en een rijke chat-UI met whiteboard, TTS, kladblaadje, zinsontleding en PDF-werkbladen.

## Milestone Summary

**Key Decisions:**

- react-to-print v3 over @react-pdf/renderer (Next.js App Router compatibility)
- Text injection over RAG for leerstof (simpler at MVP scale)
- Prompt caching architecture (static base first) — ~90% cost reduction
- Client-side router.push() over server-side redirect() (auth cookie preservation)
- profiles.id vs user_id — strict separation enforced throughout codebase
- clip-path over SVG stroke-dashoffset (Android GPU performance)
- Neural TTS (OpenAI tts-1-hd, nova stem) over browser TTS (quality)
- Rule-based study plan over AI-generated (deterministic, no cost)
- Hardcoded /nl/api/tutor/tts path — Next.js locale-route registered for all locales

**Issues Resolved:**

- Auth trigger mismatch (user_id vs id columns)
- Cookie loss on server-side redirect after OAuth
- Assessment double-fire (finishAssessment idempotency)
- Whiteboard regex g-flag bug (shared module-level regex)
- ZinsontledingPanel hasSpecialBlocks guard missing
- KaTeX external CSS bundler issue in client components
- tsconfig.json Deno URL imports breaking Next.js build

**Technical Debt Deferred:**

- SQL migrations 008, 009, 011, 012 still need manual run in Supabase SQL Editor (prod)
- "genoeg geoefend" in existing DB sessions (old context) — new sessions work correctly
- Phase 01: only 01-01-SUMMARY.md created (02, 03 plans executed but not documented)
