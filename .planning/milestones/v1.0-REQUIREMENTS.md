## v1.0 Requirements

### Authenticatie & Accounts

- [x] **AUTH-01**: Ouder kan account aanmaken via email en wachtwoord — v1.0 ✓
- [x] **AUTH-02**: Ouder kan inloggen via Google OAuth — v1.0 ✓
- [x] **AUTH-03**: Ouder kan inloggen via Facebook OAuth — v1.0 ✓
- [x] **AUTH-04**: Ouder sessie blijft actief na browser refresh — v1.0 ✓
- [x] **AUTH-05**: Ouder kan meerdere kindprofielen aanmaken (voornaam, klas 1-6, leeftijd) — v1.0 ✓
- [x] **AUTH-06**: Ouder geeft bij registratie toestemming voor AI-bijles via verplicht vinkje — v1.0 ✓

### AI Tutoring - Koko

- [x] **TUTOR-01**: Kind kan een vak kiezen en een bijlessessie starten met Koko — v1.0 ✓
- [x] **TUTOR-02**: Koko begeleidt Socratisch — geeft nooit direct het antwoord — v1.0 ✓
- [x] **TUTOR-03**: Koko spreekt Nederlands als instructietaal, schakelt naar Papiamento/Spaans — v1.0 ✓
- [x] **TUTOR-04**: Koko gebruikt Arubaanse context in voorbeelden (Hooiberg, Florins, Shoco, etc.) — v1.0 ✓
- [x] **TUTOR-05**: Koko geeft directe feedback op antwoorden (goed/fout + uitleg) — v1.0 ✓
- [x] **TUTOR-06**: Koko past moeilijkheidsgraad aan op basis van prestaties — v1.0 ✓
- [x] **TUTOR-07**: Platform ondersteunt 3 kernvakken + 3 zaakvakken — v1.0 ✓
- [x] **TUTOR-08**: Koko genereert printbare PDF-werkbladen na succesvolle oefeningen — v1.0 ✓ (Phase 6)
- [x] **TUTOR-09**: Kind kan huiswerk maken met ondersteuning van Koko via de app — v1.0 ✓ (Phase 6)
- [x] **TUTOR-10**: Kind kan gericht hiaten bijwerken per kernvak — v1.0 ✓ (Phase 6)
- [x] **TUTOR-11**: Kernvakken volgen de Arubaanse Kerndoelen en leerlijnen voor klas 1-6 — v1.0 ✓ (Phase 6)
- [x] **TUTOR-12**: Leerkracht/bijlesmeneer kan leerstof voor zaakvakken handmatig invoeren of als PDF uploaden — v1.0 ✓ (Phase 6)

### Beginsituatietoets

- [x] **TOETS-02**: Toets past moeilijkheid aan op basis van antwoorden (adaptief) — v1.0 ✓ (Phase 5)
- [x] **TOETS-03**: Na afloop wordt een startniveau bepaald per vak — v1.0 ✓ (Phase 5)

### Ouderportaal

- [x] **OUDER-01**: Ouder ziet een dashboard met voortgang per kind, per vak — v1.0 ✓ (Phase 7)
- [x] **OUDER-02**: Ouder kan kindprofielen bewerken en toevoegen — v1.0 ✓ (Phase 7)
- [x] **OUDER-03**: Ouder kan abonnementsstatus en betalingsgeschiedenis bekijken — v1.0 ✓ (Phase 7)
- [x] **OUDER-04**: Ouder ontvangt wekelijks een automatisch gegenereerd voortgangsbericht per kind — v1.0 ✓ (Phase 7)
- [x] **OUDER-05**: Ouder kan een vakantierooster bekijken met Arubaanse schoolvakanties — v1.0 ✓ (Phase 7)

### Betalingen

- [x] **PAY-01**: Ouder kan betalingsbewijs uploaden (foto bankovermaking) — v1.0 ✓ (Phase 3)
- [x] **PAY-02**: Ouder kan contante betaling aanvragen — v1.0 ✓ (Phase 3)
- [x] **PAY-03**: Toegang tot platform is gekoppeld aan actief abonnement — v1.0 ✓ (Phase 3)
- [x] **PAY-04**: Ouder kan kiezen uit abonnementsperiodes: per keer, per week, per maand, of per schooljaar — v1.0 ✓ (Phase 3)

### Admin Panel

- [x] **ADMIN-01**: Admin kan openstaande betalingsverzoeken bekijken — v1.0 ✓ (Phase 3)
- [x] **ADMIN-02**: Admin kan betaling goedkeuren en abonnement activeren — v1.0 ✓ (Phase 3)
- [x] **ADMIN-03**: Admin kan overzicht bekijken van alle families en kinderen — v1.0 ✓ (Phase 7)
- [x] **ADMIN-04**: Admin ontvangt melding (flag) als een kind 3x achtereen een concept niet begrijpt — v1.0 ✓ (Phase 7)
- [x] **ADMIN-05**: Admin kan het vakantierooster beheren (schoolvakanties Aruba) — v1.0 ✓ (Phase 7)
- [x] **ADMIN-06**: Admin kan geuploade leerstof van leerkrachten bekijken en beheren — v1.0 ✓ (Phase 6)

### Drietalige Interface

- [x] **I18N-01**: Gebruiker kan de UI schakelen tussen Nederlands, Papiamento en Spaans — v1.0 ✓ (Phase 1)
- [x] **I18N-02**: Alle UI-teksten, labels en navigatie zijn beschikbaar in drie talen — v1.0 ✓ (Phase 1)

### Privacy & Beveiliging

- [x] **PRIV-01**: Platform slaat alleen strikt noodzakelijke data op (voornaam, klas, voortgang) — v1.0 ✓ (Phase 1)
- [x] **PRIV-02**: Privacyverklaring is beschikbaar als pagina op het platform — v1.0 ✓ (Phase 1)
- [x] **PRIV-03**: Alle API-sleutels staan in .env en zijn uitgesloten via .gitignore — v1.0 ✓ (Phase 1)
- [x] **PRIV-04**: Alle database-tabellen hebben Row Level Security (RLS) ingeschakeld — v1.0 ✓ (Phase 1)

### Voortgang & Monitoring

- [x] **PROG-01**: Voortgang wordt bijgehouden per kind, per vak, per niveau — v1.0 ✓ (Phase 5)
- [x] **PROG-02**: Kind ziet eigen voortgang met visuele indicatoren — v1.0 ✓ (Phase 5)
- [x] **PROG-03**: Koko geeft een flag als een kind 3x achtereen vastloopt op een concept — v1.0 ✓ (Phase 5)
