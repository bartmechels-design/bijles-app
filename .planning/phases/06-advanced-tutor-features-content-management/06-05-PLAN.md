  <how-to-verify>
    Start the dev server first:
    ```bash
    cd aruba-leren && npm run dev -- -H 0.0.0.0
    ```

    **STAP 1: Voer SQL migratie 008 uit**
    - Open Supabase Dashboard (supabase.com → your project → SQL Editor)
    - Open bestand: `aruba-leren/supabase/migrations/008_leerstof_tables.sql`
    - Kopieer de volledige inhoud en voer uit in de SQL Editor
    - Verwacht: geen errors, "Success. No rows returned"

    **STAP 2: Test Hiaten Selectie**
    - Open http://localhost:3000/nl/tutor/[childId]/rekenen
    - Verwacht: boven het chatvenster verschijnen chip-knoppen zoals "Breuken", "Tafels", etc.
    - Klik op "Breuken basis" — chip wordt violet
    - Stuur een eerste bericht ("Hoi Koko")
    - Verwacht: chips verdwijnen, Koko reageert met een vraag specifiek over breuken (bijv. "Wat is een breuk eigenlijk?")
    - Optioneel: test URL param door `/nl/tutor/[childId]/rekenen?hiaat=breuken_basis` te openen — "Breuken basis" chip moet al geselecteerd zijn

    **STAP 3: Test PDF Werkblad**
    - Voer een Rekenen sessie uit tot Koko een [OPDRACHT] blok stuurt (of vraag handmatig: "Geef me een oefening")
    - Verwacht: het OPDRACHT blok verschijnt als een oranje gekleurde kaart in de chat
    - Na 3+ berichten van Koko: een "Print Werkblad" knop verschijnt in de toolbar (bovenaan)
    - Klik op "Print Werkblad"
    - Verwacht: browser print-dialoog opent met A4 werkblad, kindnaam, datum, en de opdrachten als nummerde kaarten
    - Sluit het print-dialoog (hoeft niet echt afgedrukt te worden)

    **STAP 4: Test Leerstof Upload**
    - Ga naar http://localhost:3000/nl/admin/leerstof (log in als admin account)
    - Verwacht: pagina toont "Leerstof Beheer" met upload formulier
    - Selecteer vak "Geschiedenis", vul titel in "Test Leerstof - Caquetio", kies modus "Tekst invoeren"
    - Plak tekst: "De Caquetio waren de oorspronkelijke bewoners van Aruba. Zij leefden van vissen en jagen. In 1499 kwamen de Spanjaarden voor het eerst naar Aruba."
    - Klik "Leerstof opslaan" — verwacht: groene succesmelding met aantal characters
    - Open vervolgens een Geschiedenis sessie voor een kind: http://localhost:3000/nl/tutor/[childId]/geschiedenis
    - Verwacht: Koko stelt in zijn eerste reactie een vraag die verband houdt met de Caquetio of die aansluit op de geüploade tekst

    **STAP 5: Test Huiswerk Modus**
    - Open http://localhost:3000/nl/tutor/[childId]/taal
    - Klik ZONDER iets te typen op de camera knop (foto upload)
    - Kies een willekeurige afbeelding van je computer (bijv. screenshot van een tekst)
    - Verwacht: blauwe "Huiswerk Hulp actief" banner verschijnt boven de chatberichten
    - Verwacht: na ~600 milliseconden wordt automatisch een bericht verstuurd met de foto en tekst "Kun je mijn huiswerk bekijken?"
    - Verwacht: Koko analyseert de afbeelding en beschrijft wat hij ziet, stelt dan een Socratische vraag
